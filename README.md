## 核心理念

DCE Protocol 是当前最核心的第一性原则：
- **D**iscuss（讨论）→ 主动挖掘信息，而非被动响应
- **C**onfirm（确认）→ 等待明确批准，避免误解执行
- **E**xecute（执行）→ 带着完整上下文精准行动

它不是一套零散技巧，而是一种组织与 AI 协作的工作方式。

## 为什么开源

我们相信：
1. 方法比工具更有价值，但工具是方法最好的广告
2. 理论是根，技能是叶，双仓分层才能形成稳定资产矩阵
3. AI 时代的组织管理需要可传播、可执行、可复用的方法论
